#  Moviefy

Moviefy is a full-stack web application for cinema ticket booking.
It combines a **Spring Boot + JPA + PostgreSQL backend** with a **React frontend** to deliver a smooth and practical movie-going experience.  

## Features
- Browse movies and showtimes  
- Interactive seat selection (with hold and expiration)  
- Secure user registration, login, and role-based access (admin / customer)  
- Admin tools for managing movies, showtimes, and users  
- Import/export data in XML with XSD validation + HTML transformations  
- PDF ticket generation with QR codes  
- Email confirmations for bookings  
- Dashboard with statistics and charts for admins  

## Tech Stack
- **Backend:** Spring Boot, Spring Data JPA, Spring Security, JavaMail, iText  
- **Frontend:** React, Axios, React Router, Context API  
- **Database:** PostgreSQL
- **Other:** XML (JAXB, XSD, XSLT), PDF, REST API
